# tatsukikitamura

Config files for my GitHub profile.

## 概要
このリポジトリは私の GitHub プロフィール用の設定ファイル（dotfiles・設定スクリプト等）をまとめたものです。主に開発環境の自動化や見た目の調整に使う設定を管理しています。

## 主な言語
- Python (100%)

## 機能
- 個人の設定ファイルの管理
- 開発環境の自動化スクリプト

## 動作環境
- OS: macOS / Linux / Windows
- Python >= 3.8

## インストール方法
1. リポジトリをクローン
```bash
git clone https://github.com/tatsukikitamura/tatsukikitamura.git
cd tatsukikitamura
```
2. 必要に応じてスクリプトや設定を実行・適用してください。
